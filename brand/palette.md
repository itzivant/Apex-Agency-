# Apex Marketing Agency — Brand Color Palette

## Primary Colors
| Name | Hex | Usage |
|---|---|---|
| Navy | `#0A1628` | Backgrounds, headers, badges, nav |
| Electric Blue | `#2563EB` | CTAs, links, highlights, logo mark, accents |
| White | `#FFFFFF` | Body text on dark, card backgrounds, icons |

## Supporting Colors
| Name | Hex | Usage |
|---|---|---|
| Slate | `#1E3A5F` | Secondary dark backgrounds, dividers, hover on navy |
| Sky | `#3B82F6` | Hover state for Electric Blue elements |
| Light Gray | `#F1F5F9` | Page backgrounds, section dividers on light layouts |
| Mid Gray | `#64748B` | Secondary body text, captions, placeholders |

## Status / Functional Colors (dashboard only — not brand elements)
| Name | Hex | Usage |
|---|---|---|
| Success Green | `#10B981` | Signed/active/sent status |
| Warning Amber | `#F59E0B` | Pending approval, follow-up needed |
| Danger Red | `#EF4444` | Overdue, blocked, disqualified |
| Info Blue | `#2563EB` | New / in-progress (reuses Electric Blue) |

## Usage Rules
1. **Never** put Electric Blue text on a Navy background without a size ≥14px — contrast ratio is marginal for small text.
2. Navy + White is the primary hero combination. Electric Blue is for interactive elements only.
3. Don't use Electric Blue as a section background — it's an accent, not a base color.
4. Status colors are functional (dashboard/badges) only. Keep them out of marketing materials.
5. On white backgrounds, use Navy for headlines and Mid Gray for body copy.

## Typography
| Role | Font | Weight | Size |
|---|---|---|---|
| Headings | Inter or DM Sans | 700 | 24–48px |
| Section labels | Inter | 700 | 11–12px, uppercase, tracked 0.12em |
| Body | Inter | 400 | 16px base |
| Data / code | JetBrains Mono | 400 | 14px |
| Tagline | Inter | 400 | 11px, uppercase, tracked 0.18em |

## Logo Clearspace
- Minimum clearspace: 16px on all sides of the badge
- Minimum size: 160px wide (don't scale below this)
- Use `brand/logo.svg` — it has a dark variant (`#logo-dark`)
- On white backgrounds: place the full dark badge; do not remove the navy rectangle
